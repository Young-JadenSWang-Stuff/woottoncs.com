Formatting Strings

- String formatting uses the format method called on the String class
- The first parameter is a string parameter that tells the method how to format each String

- To tell the method which piece of information in your format parameter you need to use the % symbol
- Lets say you have 3 strings to format: Name, Date, Period
- The format method is expecting 3 segments in our format parameter starting with the % symbol
- Name is a String, Date is a String, Period is a number (integer)

- The template for each format piece is percent sign (%)
- A sign to indicate alignment (- for left align, nothing for right align)

- Strings are represented with an S
- Characters are represented with a C
- Integers are represented with a D
- Doubles are represented with a F

- Lets say we want to align everything to the left with 20 spaces allocated for each element
- First, we need to do the first two strings, %-20s %-20s
- Then for the third parameter we need a %-20d

- To reuse our format many times and keep our code consice, we should save the format to a string
- String format = "%-20s %-20s %-20d";

- parameters are separated with a comma
- String.format(format, "Chicken", "11/11/11", 38);
Writing to a File

- First, write a try catch statement to create the file
- System.exit(-1); is to be used when you are in way too deep and have no way to return out, do not use this extensively
- Write another try catch to begin writing to a the file
- Use the method write() to begin writing
- add the escape sequences "\r\n" to create a line break (go to the next line)
- After you finish writing, use the close() method to indicate that you are done with the file, this effectively saves the file

Resizing Arrays

- Resizing an array requires the recopying of the entire old array and placing it into a new larger one
- Create a temporary array an element larger
- Loop through your original array and copy all data into the second array
- Return the second array
